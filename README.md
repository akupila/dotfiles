# dotfiles

My dotfiles.

## yadm

```bash
brew install yadm
```

## neovim

```bash
brew install neovim
```

## vim-plug

Plugin manager for vim

```bash
curl -fLo ~/.config/nvim/autoload/plug.vim --create-dirs https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

## Install dotfiles

```bash
yadm clone git@github.com:akupila/dotfiles.git
```
