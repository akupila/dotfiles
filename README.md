# dotfiles

## Install yadm

```bash
brew install yadm
```

## Install Vundle

```bash
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

## Install dotfiles

```bash
yadm clone git@github.com:akupila/dotfiles.git
```
